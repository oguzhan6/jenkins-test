About This Project

This project is a demonstration of a modern web application built with a containerized infrastructure and deployed via a CI/CD pipeline.

Core Technologies

  • Backend: PHP 7.4 running on an Apache server.
  • Database: MySQL, providing persistent data storage for the guestbook.
  • Frontend: Vanilla JavaScript (ES6+), HTML5, and CSS3 for an interactive user experience without frontend frameworks.
  • Containerization: Docker and Docker Compose, allowing the entire application and its services to run in isolated, reproducible environments.

DevOps & Monitoring

  • CI/CD Pipeline: Jenkins, which automates the build and deployment process directly from the GitHub repository.
  • Monitoring Stack:
    • Prometheus: Scrapes and stores time-series metrics from application services.
    • Grafana: Provides rich, interactive dashboards for visualizing the metrics collected by Prometheus.
    • cAdvisor: An agent that exposes detailed container metrics (CPU, Memory, Network) to Prometheus.

Features

  • Interactive Guestbook: A real-time, single-page-feel guestbook that allows users to post and view messages without page reloads, using JavaScript's Fetch API to communicate with the PHP backend.
  • Live System Stats: A dashboard that displays live metrics from the server, gathered by a custom PHP API endpoint.
